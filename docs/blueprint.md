# **App Name**: AttendanceEase

## Core Features:

- Class Management: Create and manage class schedules with details like time, date, and location.
- Bulk Student Upload: Upload student details in bulk via CSV or Excel files to quickly populate class rosters.
- Attendance Marking: Mark attendance for individual students or use a bulk attendance marking feature.
- Attendance Report Generation: Generate detailed attendance reports with customizable filters (e.g., date range, class, student).
- Attendance Percentage Calculation: Automatically calculate the attendance percentage for each student in every course.
- Report Download: Download attendance reports in common formats like CSV or PDF, using templates to preserve special layouts.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6), evoking trust and reliability for educational data.
- Background color: Light, airy off-white (#F5F5F5), provides a clean and unobtrusive backdrop.
- Accent color: Gentle green (#81C784), symbolizing growth and positivity.
- Body and headline font: 'PT Sans', a humanist sans-serif, offering both modernity and warmth, for a readable and inviting experience.
- Simple, clear icons for actions like uploading, downloading, and filtering.
- Responsive layout design to adapt seamlessly to both mobile and laptop screens.
- Subtle transition animations to provide a smooth user experience.
